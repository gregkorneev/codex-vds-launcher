# Codex CLI Launcher v1.0.0

## Русский

**Codex CLI Launcher v1.0.0** — первый стабильный релиз Launcher. Он объединяет все функции, созданные и проверенные в Developer Beta 1–9, в стабильной сборке для macOS на Apple Silicon и Windows x64.

### Полный набор возможностей v1

- Запуск установленного Codex CLI локально в выбранной папке проекта.
- Запуск Codex CLI на VDS через пользовательский OpenSSH alias без хранения паролей, ключей и токенов.
- Встроенный терминал xterm.js на `node-pty`, корректно подстраивающийся под окно и боковые панели.
- Отдельные локальные и VDS-проекты с безопасным добавлением и удалением записей без удаления файлов проекта.
- Одновременные сессии для нескольких проектов, переключение, остановка, перезапуск, очистка и открытие внешнего терминала.
- Локальная история терминала по проектам и выгрузка последнего запуска выбранной сессии в читаемый Markdown.
- Быстрые промпты и наборы команд в едином редактируемом списке «Быстрые действия».
- Импорт локальных Markdown-инструкций через нативный выбор файла или drag-and-drop.
- Синхронизация управляемых инструкций `AGENTS.md`.
- Переключение русского и английского языка, светлой и тёмной темы, а также акцентного цвета.
- Сворачиваемые панели и разделы с сохранением состояния.
- Профиль Codex с именем, email, типом подписки, оставшимися лимитами, временем сброса и дополнительными сбросами — без чтения токенов.
- Компактные read-only карточки состояния VDS, SSH, Codex и опционального `codex-vpn`.
- Меню приложения на macOS, tray-меню на Windows и быстрый выбор локального/VDS-режима.
- GitHub Releases, beta/latest каналы обновлений и безопасный ручной DMG-переход для ad-hoc macOS-сборок.
- Первый запуск и экран после обновления с инструкцией подключения и обзором изменений.
- MIT License, NOTICE, third-party notices и команды аудита лицензий.

### Путь от Beta 1 до Beta 9

- Появилась раскрываемая панель оформления, светлая/тёмная тема и выбор акцентного цвета.
- README и документация стали двуязычными.
- Подготовлена первая developer beta сборка для macOS.

- «Конфигурация VDS» и «Статус сервера» стали сворачиваемыми.

- Быстрые промпты стало можно добавлять, редактировать и удалять в приложении.
- Добавлены MIT License, NOTICE, third-party notices и license audit scripts.

#### Developer Beta 3

- Уменьшены отступы вокруг свёрнутых секций и унифицировано их оформление.
- Onboarding и справка стали подробно объяснять установку Codex CLI на VDS и подключение через Launcher.
- В релизную матрицу добавлен Windows x64 NSIS installer.
- Уточнена поставка macOS DMG и назначение `codex-vpn`.

#### Developer Beta 4

- Добавлена система обновлений через GitHub Releases.
- Beta-сборки получили prerelease-канал, а стабильные сборки — канал `latest`.
- В интерфейсе появилась проверка и установка обновлений с понятными статусами.
- После обновления Launcher показывает приветственный экран текущей версии.
- Добавлены metadata-файлы обновления для macOS и Windows; Beta 4.1 проверила и уточнила этот путь обновления.

#### Developer Beta 5

- Добавлены два полноценных способа запуска: локальный Codex CLI и Codex CLI на VDS.

#### Developer Beta 6

- Приложение переименовано из Codex VDS Launcher в **Codex CLI Launcher**.
- В «Настройках» появился явный выбор «На этом компьютере / На VDS через SSH».
- Локальный режим стал скрывать VDS-конфигурацию и серверные карточки.
- Быстрые промпты и команды объединены в один раздел «Быстрые действия».
- Проекты, Обновления, Локальные MD и Быстрые действия стали сворачиваемыми и сохраняют состояние.
- Welcome-экран стал объяснять оба режима и показывать изменения после каждого обновления.

#### Developer Beta 7

- Пользовательские проекты стало можно безопасно удалять из Launcher без удаления папок и файлов.
- Быстрые промпты и команды объединены в один визуальный список с одинаковой вставкой в активную сессию.
- Переключение режима добавлено в macOS-меню и Windows tray.
- Появился раскрываемый профиль Codex с аккаунтом, тарифом, остатком лимитов и временем сброса.
- Профиль научился работать с локальным Codex CLI и выбранной VDS без чтения и отображения токенов.
- Обновлены иконки macOS, Windows, tray и интерфейса.

#### Developer Beta 8

- Исправлено наложение встроенного терминала на правую панель; размер теперь отслеживается по реальному контейнеру.
- Профиль Codex стал отдельно показывать имя, email, тип подписки, лимиты и время их обновления.
- Добавлен безопасный fallback имени из части email до `@`.
- Добавлено отображение доступных дополнительных сбросов лимита.
- Обновлены иконки приложения для macOS, Windows и интерфейса Launcher.

#### Developer Beta 9

- Возле каждой сессии появилась кнопка выгрузки истории последнего запуска.
- Launcher сохраняет историю выбранной сессии через нативный Save dialog в `<сессия>-history.md`.
- Markdown содержит проект, путь, время выгрузки и очищенную от ANSI-последовательностей историю терминала.
- Последний запуск хранится отдельно по проектам и сохраняется между перезапусками Launcher.
- Release workflow унифицирован для Beta prerelease и стабильных semver-тегов macOS/Windows.

### Артефакты и обновления

- macOS Apple Silicon: DMG, ZIP и `latest-mac.yml`.
- Windows x64: NSIS EXE и `latest.yml`.
- Стабильная версия использует канал обновлений `latest`.
- macOS-приложение подписано ad-hoc и не notarized, поэтому обновление на macOS открывает официальную страницу релиза для ручной установки DMG.

## English

**Codex CLI Launcher v1.0.0** is the first stable Launcher release. It combines everything built and tested from Developer Beta 1 through Developer Beta 9 in stable packages for Apple Silicon Macs and Windows x64.

### Complete v1 feature set

- Run an installed Codex CLI locally in a selected project folder.
- Run Codex CLI on a VDS through a user-managed OpenSSH alias without storing passwords, keys, or tokens.
- Embedded xterm.js terminal backed by `node-pty`, sized to the actual window and side panels.
- Separate local and VDS projects with safe entry removal that never deletes project files.
- Multiple project sessions with switch, stop, restart, clear, and external-terminal actions.
- Per-project local terminal history and readable Markdown export for the selected session's latest run.
- Quick prompts and command sets in one editable Quick actions list.
- Local Markdown instruction import through a native picker or drag-and-drop.
- Managed `AGENTS.md` instruction synchronization.
- Russian/English UI, light/dark themes, and accent colors.
- Collapsible panels and sections with persistent state.
- Codex profile with name, email, subscription type, remaining limits, reset times, and extra resets without reading tokens.
- Compact read-only VDS, SSH, Codex, and optional `codex-vpn` status cards.
- macOS application menu, Windows tray menu, and quick local/VDS mode selection.
- GitHub Releases, beta/latest update channels, and a safe manual DMG path for ad-hoc macOS builds.
- First-launch and post-update guidance with connection steps and release highlights.
- MIT License, NOTICE, third-party notices, and license-audit commands.
