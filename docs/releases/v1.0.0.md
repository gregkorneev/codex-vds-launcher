# Codex CLI Launcher v1.0.0

## Русский

В Codex CLI Launcher v1 представлены следующие функции:

- Запуск установленного Codex CLI локально в выбранной папке проекта.
- Запуск Codex CLI на VDS через пользовательский OpenSSH alias без хранения паролей, ключей и токенов.
- Встроенный терминал xterm.js на `node-pty`, автоматически подстраивающийся под размер окна и боковых панелей.
- Отдельные списки локальных и VDS-проектов с безопасным добавлением и удалением записей без удаления файлов проекта.
- Одновременная работа с несколькими проектными сессиями: переключение, остановка, перезапуск, очистка и открытие во внешнем терминале.
- Локальное сохранение истории терминала отдельно для каждого проекта.
- Выгрузка истории последнего запуска выбранной сессии в Markdown через нативный диалог сохранения.
- Очистка экспортируемой истории от ANSI-последовательностей и добавление в Markdown сведений о проекте, пути и времени выгрузки.
- Редактируемый список «Быстрые действия», объединяющий быстрые промпты и наборы команд.
- Импорт локальных Markdown-инструкций через нативный выбор файла или drag-and-drop.
- Синхронизация управляемых инструкций `AGENTS.md`.
- Переключение русского и английского языка интерфейса.
- Светлая и тёмная темы с выбором акцентного цвета.
- Сворачиваемые панели и разделы с сохранением их состояния.
- Профиль Codex с именем, email, типом подписки, оставшимися лимитами, временем сброса и дополнительными сбросами — без чтения токенов.
- Компактные read-only карточки состояния VDS, SSH, Codex и опционального `codex-vpn`.
- Меню приложения на macOS, tray-меню на Windows и быстрый выбор локального/VDS-режима.
- Проверка обновлений через GitHub Releases с отдельными каналами `beta` и `latest`.
- Приветственный экран первого запуска, инструкция по настройке VDS и обзор изменений после обновления.

## English

Codex CLI Launcher v1 introduces the following features:

- Run an installed Codex CLI locally in a selected project folder.
- Run Codex CLI on a VDS through a user-managed OpenSSH alias without storing passwords, keys, or tokens.
- Embedded xterm.js terminal backed by `node-pty` and automatically sized to the window and side panels.
- Separate local and VDS project lists with safe entry creation and removal that never deletes project files.
- Multiple project sessions with switch, stop, restart, clear, and external-terminal actions.
- Local terminal history stored separately for every project.
- Export the selected session's latest-run history to Markdown through a native Save dialog.
- Strip ANSI sequences from exported history and include project, path, and export-time details in the Markdown file.
- One editable Quick actions list combining quick prompts and command sets.
- Import local Markdown instructions through a native file picker or drag-and-drop.
- Synchronize managed `AGENTS.md` instructions.
- Switch between Russian and English interface languages.
- Light and dark themes with selectable accent colors.
- Collapsible panels and sections with persistent state.
- Codex profile with name, email, subscription type, remaining limits, reset times, and extra resets without reading tokens.
- Compact read-only VDS, SSH, Codex, and optional `codex-vpn` status cards.
- macOS application menu, Windows tray menu, and quick local/VDS mode selection.
- Check for updates through GitHub Releases with separate `beta` and `latest` channels.
- First-launch welcome screen, VDS setup guide, and post-update release highlights.
